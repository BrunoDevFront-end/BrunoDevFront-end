## 🔥 Projeto em destaque

<table>
<tr>
<td width="100%" valign="top">

# 🚀 Anonymous Feedback SaaS

Estou desenvolvendo uma plataforma **SaaS de feedbacks anônimos**, criada para permitir que empresas coletem opiniões de seus usuários, clientes ou colaboradores de forma simples e acessível.

O projeto está sendo desenvolvido com foco em práticas utilizadas no desenvolvimento de aplicações reais.

<br>

### ✨ Principais funcionalidades

🔐 **Autenticação com JWT**

🛡️ **Middleware de autenticação e autorização**

🔑 **Criptografia segura de senhas**

✅ **Validação de dados com Zod**

📝 **Criação e gerenciamento de formulários**

⭐ **Sistema de avaliação por estrelas**

📊 **Dashboard com estatísticas**

🗄️ **PostgreSQL + Prisma ORM**

🔄 **API REST com Node.js e Express**

📱 **Interface responsiva**

<br>

> Este projeto tem sido minha principal experiência prática para aprofundar conhecimentos em desenvolvimento Full Stack, segurança, modelagem de banco de dados, APIs, autenticação e arquitetura de aplicações.
